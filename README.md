# Weather App

Este proyecto es una aplicación iOS meteorológica, la aplicación mostrará la temperatura de una ubicación. Para este proyecto se usó la API de OpenWeather para obtener datos meteorológicos. También se usó el lenguaje de programación Swift con el framework SwiftUI.
## Caracteristicas

- SwiftUI
- RxSwift
- Alamofire
- Moya
- GoogleMaps
- GooglePlaces
- Lottie
- GRDB


## Instalación

Instalar WeatherApp con CocoaPods

```bash
  cd WeatherApp
  pod install
  open Weather\ App.xcworkspace
```
